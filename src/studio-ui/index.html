群组名称
协作模式
⚔️
Debate 辩论模式
多个 Agent 围绕一个话题正反方辩论，最终由裁判 Agent 综合各方观点给出结论。
💡 适合：决策分析、方案对比、风险评估
📤 产出：多角度分析报告 + 最终建议
🗳️
Voting 投票模式
每个 Agent 独立给出判断和投票，汇总统计后输出多数意见和少数意见。
💡 适合：质量评审、内容审核、多人打分
📤 产出：投票结果 + 各方理由汇总
🔗
Pipeline 流水线模式
Agent A 的输出自动传给 Agent B 作为输入，逐步处理和加工，像工厂流水线。
💡 适合：内容创作、数据处理、翻译校对
📤 产出：经过多轮加工的最终成果
🏛️
Hierarchy 层级模式
一个主管 Agent 拆解任务分配给下属 Agent，收集结果后汇总报告。
💡 适合：项目管理、调研收集、并行执行
📤 产出：汇总报告 + 各子任务结果
🧠
Shared Memory 共享记忆模式
所有 Agent 共享一个知识空间，各自贡献信息，互相读取和补充，持续积累。
💡 适合：知识构建、团队学习、长期项目
📤 产出：持续进化的共享知识库
